# 🔥 星火报销助手

> 智能出差报销工具 —— 截图自动识别，一键生成报账单

专为差旅报销设计的手机 App，拍照/截图自动识别消费信息，一键生成标准报账单和发票汇总文档，告别手工填表。

---

## ✨ 主要功能

- 📸 **智能识别** — 拍照/选图自动 OCR 识别金额、日期、时间、消费类型
- 🍚 **自动分类** — 按时段自动区分早/午/晚餐，智能计算补助限额
- 🚕 **交通补助** — 支持"每日可用"和"仅首末日"两种模式
- 📋 **一键生成** — 自动生成差旅补助申领确认表（Word 格式）
- 🖼️ **发票汇总** — 所有发票图片自动排版为 Word 文档
- 📅 **按天归档** — 发票按天分页，方便审核人逐日核对
- 📝 **备注标注** — 可在发票图片底部添加红色说明文字
- 📤 **一键发送** — 报账单 + 发票图集打包为 ZIP，微信/邮箱/钉钉一键分享
- 🔄 **OTA 更新** — 应用自动检测更新，无需重新下载安装

## 📖 使用说明

### 一、首次使用：填好设置

打开 App → 底部 **「设置」** 页：

1. **姓名** — 会显示在报账单上，必填
2. **部门** — 默认「信息技术研究所」，按需修改
3. **交通补助方式** — 每日可用 / 仅首末日
4. **每页图片数** — 发票汇总文档每页放几张（默认 6 张）

### 二、创建出差计划

在 **「设置」** 页新建出差计划 → 选好日期 → 点 **「激活」**。

### 三、导入消费截图

到 **「发票」** 页 → 点 **「📷 从相册中导入消费截图」**：

- 支持批量导入，App 自动识别金额、日期、类型
- 识别成功的直接入库；有疑问的进入待确认队列手动核对
- **检测到退款信息的截图会自动标红提醒**

> 💡 建议用支付详情页/订单详情页截图（能看清金额和具体时间），识别最准。

### 四、核对与预览

- **修改**：每条记录可改类型、金额、日期、时间
- **备注**：可在发票底部加红色说明文字，字体大小可调
- **详情预览**：点击记录查看原图+提取信息，左右滑动切换
- **预览页**：一眼看清每天的消费和补助，**补助金额可手动调整**

### 五、生成报账单

点 **「📋 生成报账单」** → 选择报销时间 → 生成。页面会列出：

| 文件  | 说明  |
| --- | --- |
| 📝 差旅补助申领确认表.docx | 报账单 |
| 🖼️ 发票图片汇总.docx | 紧凑排版，方便打印 |
| 📅 发票图片(按天).docx | 每天一页，方便审核 |
| 📦 ZIP 压缩包 | 包含以上三个文件 |

点击文件可预览，底部有 **删除** 和 **发送** 按钮。

## ❓ 常见问题

**Q：识别错了怎么办？**
点「修改」按钮手动改正，报账单会同步更新。

**Q：传错了图怎么办？**
点「删除」按钮删掉即可。

**Q：换手机数据会丢吗？**
数据存在手机本地，建议先生成报账单发到邮箱备份。

**Q：报账单金额和实际消费不一样？**
报账单显示的是按补助规则封顶后的可报金额，不是实际消费总额。
