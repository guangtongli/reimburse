# 星火报销助手 v1.0.0  智能出差报销助手，拍照即识别发票，一键生成报账单 、影像文档并发送（微信、钉钉、邮件）。  ## ✨ 主要功能  - 📷 **智能识别**：拍照/选图自动 OCR 识别发票信息（金额、日期、商家、地址） - 📊 **自动汇总**：按出差计划自动归类消费，可添加备注，生成标准报账单 - 📄 **Word 报账单**：一键生成带标题的 Word 文档，格式规范 - 🖼 **发票图集**：所有发票图片自动排版为 Word 文档 - 📧 **一键发送**：报账单 + 发票图集，一键微信/钉钉/邮件发送 - 🔄 **OTA 更新**：应用自动检测更新，无需重新下载安装
